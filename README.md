# Python Projects

A collection of various Python scripts and applications, ranging from simple command line tools to full management systems.

##  Projects List

*   **Bank Account System** (`bank_account_system.py`): Simulates basic banking operations like deposits, withdrawals, and balance checks.
*   **BMI Calculator** (`bmi_calculator.py`): Calculates Body Mass Index based on user input for height and weight.
*   **Calculator** (`calculator.py`): A standard calculator for basic arithmetic operations.
*   **Contact Book** (`contact_book.py`): Manages a digital address book to save, view, and delete contacts.
*   **CSV Data Analyzer** (`csv_data_analyzer.py`): Reads and analyzes structured data from `data.csv`.
*   **Expense Tracker** (`expense_tracker.py`): Helps track daily expenses and categories.
*   **Library Management System** (`library_management_system.py`): Manages books, tracking checkouts and returns.
*   **Number Guessing Game** (`number_guess.py`): A simple game where the user guesses a randomly generated number.
*   **Password Generator** (`password_generator.py`): Generates secure, random passwords based on user preferences.
*   **Student Management System** (`student_management_system.py`): Manages student records, grades, and information.
*   **To-Do List** (`to_do_list.py`): A command line utility to add, view, and complete tasks.
*   **Unit Converter** (`unit_converter.py`): Converts different units of measurement like temperature, length, and weight.
